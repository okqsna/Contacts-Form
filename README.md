# Contact-Form
> Simple contact form that shares information into txt-file

### Used technologies
- `HTML`
- `SASS`
- `CSS(compiled with Node.js)`
- `SCSS`
- `JavaScript`
- `Node.js`

How to run this form in console:
```
npm install express
node app
```
then all information typed in forms will appear on txt-file in folder
